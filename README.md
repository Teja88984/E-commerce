# E-Commerce Order Management System (MySQL)

## Features
- Customer login with roles
- Product management with stock
- Cart and order processing
- Payment system
- OTP verification
- Reviews and ratings

## Database Tables
customers  
products  
cart  
orders  
order_items  
payments  
otp_verification  
reviews  
cart_history  

## How to Run
1. Open MySQL
2. Run ecommerce_database.sql
3. Database will be created automatically

## Backup
Run ecommerce_backup.sql anytime to reset the system
